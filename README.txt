Ray tracing experiment. 
Ray tracing is a technique for generating an image by tracing the path of light through pixels in an image plane
and simulating the effects of its encounters with virtual objects.
Developed using C# and 3-D vector types from WPF.

Application is not optimized. 
Launch takes ~1 minute.
